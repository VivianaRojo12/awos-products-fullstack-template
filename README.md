# AWOS - Practica Full-Stack (Products) - GitHub
Este repositorio es una practica sencilla para implementar lo que se disena en un API Spec + OpenAPI (JSON):

Incluye:
- API (Node.js + TypeScript + Express) para el recurso **Products**
- Cliente (Next.js + TypeScript) que consume la API
- Contrato OpenAPI 3.x en JSON: `openapi.json`
- Postman: environment + collection (tests basicos)
- Ejemplos JSON (request/response/error) en `/examples`

---

## Requisitos
- Node.js 18+
- NPM
- Postman
